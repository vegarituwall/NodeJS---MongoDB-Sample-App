Wall Functions

1. To create a wall go to: /create-wall/id => creates new wall with id: id.

2. To view a wall document: /walls/id

3. Update or add values to a document: /collection/walls { "_id": "id", "new_fields": {any} }
4. Set object values for root field (/walls/:id/set/:object_name). Example: "/walls/61/set/settings"
5. Add item to collection (/walls/:id/push/:collection). Example: "/walls/61/push/tiles"
6. Remove item form collection (/walls/:id/remove/:collection). Example: "/walls/61/remove/tiles"
7. Delete root fields on a wall Document (/walls/:id/unset). Example: "/walls/61/unset"
8. Update item in collection (/walls/:id/push/:collection). Example: "/walls/61/push/tiles"

Users Functions (Get, Add, Update, Remove)

9. Get user data:
/users/

10. Get one user:
/users/:fieldName/:value

11. Post values for user data (Post: /create-user Body: (data in JSON format)).
12. Update or add values to a document: /collection/users { "_id": "id", "new_fields": {any} }

13. Remove user: /remove-user/:_id (removes given _id user.)
